Lista de seleccion múltiple

Cabecera que generará el contenedor

Los listados de selección múltiple permiten agrupar listados de checkboxes en una especie de combo simulado que muestra información sobre la selección.

Para poder implementar este componente necesitamos crear un div que tenga como class el valor dropdown-link

El div principal deberá contener 3 elementos:

  • Label: Contiene la etiqueta visible para el usuario.
  • Combo: Es un span con la metainformación del componente. Deberá tener la clase css checklistflexdown-button. El contenido del contenedor será un span con class="selection" y un tag de icono i con class="icono-desplegable-simple-gran". La metainformación proporcionada por este componente será la siguiente:
    • data-flexmenu identificador del fieldset contenedor de la lista de checkboxes.
    • data-noselectiontext texto que debe utilizar el componente cuando no haya elementos seleccionados.
    • data-fullselectiontext texto que debe utilizar el componente cuando todos los elementos de la lista esten seleccionados.
    • data-partselectiontext texto que debe utilizar el componente cuando se selecciones más de un elemento pero menos que todos. Por ejemplo si le asignamos el valor "opciones", cuando seleccionemos 5 opciones de 10 tendremos el valor 5 opciones.
    • data-unica debe tener el valor y
    • data-offsets debe tener el valor 0,-1
  • Fieldset desplegable: Listado de checkboxes que apareceran al desplegar el menu . Sus estilos vienen marcados por la css a nivel de fieldset flexdropdownchecklist. El identificador del fieldset deberá tener el mismo valor que el data-flexmenu del combo. Cada elemento de la lista de checkboxes estará contenido en un label que internamente se estructurará de la siguiente forma:
    • Checkbox: Input de tipo checkbox.
    • Etiqueta: Formatearemos la etiqueta con un span con class="label".

<div class="dropdown-link">
	<label>Categoría: </label>
	<span class="checklistflexdown-button" 
				data-flexmenu="checklistflexdown" 
				data-noselectiontext="Ninguno" 
				data-fullselectiontext="Todos" 
				data-partselectiontext="seleccionados" 
				data-unica="y" 
				data-offsets="0,-1" >
			
		<span class="selection">Ninguno</span>
		<i class="icono-desplegable-simple-gran"></i>
	</span>	
	<fieldset id="checklistflexdown" class="flexdropdownchecklist">
		<label>
			<input value="a" name="optionsCheckboxList1" type="checkbox"> 
			<span class="label"> Llamada</span> 
		</label> 
		<label>
			<input value="b" name="optionsCheckboxList1" type="checkbox"> 
			<span class="label"> Chat</span> 
		</label> 
		<label>
			<input value="c" name="optionsCheckboxList1" type="checkbox"> 
			<span class="label"> Lorem</span> 
		</label> 
		<label>
			<input value="d" name="optionsCheckboxList1" type="checkbox"> 
			<span class="label"> Ipsum</span>
		</label> 
		<label>
			<input value="e" name="optionsCheckboxList1" type="checkbox"> 
			<span class="label"> dolor</span> 
		</label> 
		<label>
			<input value="f" name="optionsCheckboxList1" type="checkbox"> 
			<span class="label"> sit</span> 
		</label> 
	</fieldset>
</div>

Checklist desplegable con iconos

Cabecera que generará el contenedor

Los listados de selección múltiple permiten incluir iconos como parte de la presentación. Para ello lo único que debemos variar es el Fieldset desplegable.

  • Fieldset desplegable: Listado de checkboxes que apareceran al desplegar el menu. Sus estilos vienen marcados por la css a nivel de fieldset flexdropdownchecklist. El identificador del fieldset deberá tener el mismo valor que el data-flexmenu del combo. Cada elemento de la lista de checkboxes estará contenido en un label que internamente se estructurará de la siguiente forma:
    • Checkbox: Input de tipo checkbox.
    • Icono: Icono que se desea incluir.
    • Etiqueta: Formatearemos la etiqueta con un span con class="label".
