Botones

Botones

Los botones deberán tener la clase css btn. 


<button title="Título del botón" class="btn" >Acción</button>


Para mostrar botones destacados deberemos añadir la clase al css emphasized. 


<button title="Título del botón" class="btn emphasized">ACCION</button>


Los botones deshabilitados deberán tener el atributo disabled="disabled". 


<button title="Título del botón" class="btn" disabled="disabled">Deshabilitado</button>
