Parrilla de iconos semafóricos

Cabecera que generará el contenedor

Para hacer aparecer una parrilla de iconos semafóricos crearemos un DIV con clase semaphoric-grid. En el mismo definiremos un DIV de clase title con el título que ha de acompañar a la parrilla, y una lista con los elementos a mostrar.

La lista de elementos a mostrar la definiremos utilizando los tags estándar <ul> y <li>. A la izquierda de cada elemento haremos aparecer un semáforo verde, ambar o rojo definiendo un <i> de clase icono-semaforico-verde, icono-semaforico-ambar o icono-semaforico-rojo. 


<div class="widget-control-group-content">
	<div class="semaphoric-grid">
		<div class="title">CAJEROS</div>
		<ul>
			<li>
				<i class="icono-semaforico-ambar"></i>
