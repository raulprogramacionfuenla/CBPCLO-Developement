Separador por rangos

Título Opcional:
  • Primer elemento
  • Segundo elemento
  • Tercer elemento
  • Elemento Actual
  • Último elemento

El separador por rangos se forma mediante una lista ul con clase separador-rangos. En caso de requerir un titulo, se puede encapsular dentro del tag dl con la class css vista-por tal y como se muestra en el ejemplo.

En esta lista se puede indicar el título mediante el tag dd al mismo nivel que el tag dd que encapsula la lista ul.

Dentro del ul se especificará el orden de los elementos li de la siguiente forma: El primer elemento se designará usando la clase css primer, el elemento actual con la clase css actual y el ultimo elemento con la clase css ultimo. 


<dl class="vista-por">
	<dt>Título Opcional:</dt>
	<dd>
   		<ul class="separador-rangos">
      		<li class="primer"><a href="#">Primer elemento</a></li>
       		<li><a href="#">Segundo elemento</a></li>
       		<li><a href="#">Tercer elemento</a></li>
       		<li class="actual">Elemento Actual</li>
       		<li class="ultimo"><a href="#">Último elemento</a></li>
   		</ul>
   	</dd>
</dl>

Un grupo de pestañas irá contenido dentro de un div con estilo tabbable. Primero se deben definir las distintas pestañas que se tendrán, usando para ello un listado ul con los estilos nav nav-tabs. Definiremos un identificador para cada pestaña para enlazar con los distintos contenidos. La pestaña activa vendrá determinada por la class active.

El contenido de las pestañas se situará dentro de un div de estilo tab-content. Para cada pestaña definiremos otro div con el identificador que habremos definido previamente. Y con el estilo tab-pane. El contenido de la pestaña activa también vendrá determinada por la class active. 


<div class="tabbable">
    <ul class="nav nav-tabs">
    <li class="active"><a data-toggle="tab" href="#tab11">Pestaña 1</a></li>
    <li><a data-toggle="tab" href="#tab12">Pestaña 2</a></li>
    <li><a data-toggle="tab" href="#tab13">Pestaña 3</a></li>
  </ul>
  <div  class="tab-content">
    <div id="tab11" class="tab-pane active">
       Contenido pestaña 1
    </div>
    <div id="tab12" class="tab-pane">
       Contenido pestaña 2
    </div>
    <div id="tab13" class="tab-pane">
       Contenido pestaña 3
    </div>
  </div>
</div>

Enlaces

Enlace a otra página

Los enlaces entre páginas se realizarán utilizando la etiqueta HTML a. 


<a href="[URL]">Texto</a>


Enlace a correo electrónico

Para poder enlazar un correo se añadirá mailto: antes del correo en el atributo href de la etiqueta a. 


<a href="mailto:mail@host.com">Enlace a correo electrónico</a>

Enlace a documento 100 KB

Los enlaces a tipos de documento incluyen un icono con el logotipo del documento, el nombre del documento y el peso del documento.

Los iconos de tipos de documento proporcionados son icono-documento-excel, icono-documento-generico, icono-documento-powerpoint, icono-documento-word y icono-documento-pdf,

El peso del documento se incluirá con un tag span qye tenga la clase css document-metadata


<a href="[URL]"><i class="icono-documento-excel"></i> Nombre documento</a><span class="document-metadata">100 KB</span>
