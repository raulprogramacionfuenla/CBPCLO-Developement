Para intoducir un icono proporcionado por la arquitectura ABSIS Cloud, seguiremos las directrices marcadas por bootstramp. Para ello añadiremos un tag html <i> en nuestra página. Al tag le añadiremos una clase de estilos en el que le indicaremos el nombre del icono que queremos insertar (ver nombres en el catálogo).

Hace falta remarcar que los iconos del catálogo de iconos de ABSIS Cloud su nombre siempre empieza por el prefijo icono.

Por ejemplo, si queremos introducir el icono : añadiremos un tag <i> con una class icono-link-escenario, veamos el ejemplo:

 <i class="icono-paginador-inicio"></i>
