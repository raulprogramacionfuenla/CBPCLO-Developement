Elementos desplegables

Estilo de elementos de la Lista Desplegable

Para tener una entrada de la lista desplegable obligatoria tendremos que añadir el estilo required al de form-group a la lista desplegable estándard. 


<div class="form-group required">
  <label for="select-require" class="control-label">Lista desplegable*:</label>
  <div class="controls">
	<select id="select-require">
		<option>opcion</option>
		<option>2</option>
		<option>3</option>
		<option>4</option>
		<option>5</option>
	</select>
  </div>
</div>


Para un campo lista desplegable inhabilitado, a parte del atributo disabled en select, se deberá añadir el estilo disabled al de form-group. 


<div class="form-group disabled">
  <label for="select-disabled" class="control-label">Lista desplegable inhabilitada:</label>
  <div class="controls">
	<select id="select-disabled" disabled="disabled">
		<option>opcion</option>
		<option>2</option>
		<option>3</option>
		<option>4</option>
		<option>5</option>
	</select>
  </div>
</div>


Corregir el error

Para mostrar un error, se deberá añadir el estilo error al de form-group. Si se quiere añadir un mensaje aclaratorio, el texto deberá ser del estilo help-block. 


<div class="form-group error">
  <label for="select-error" class="control-label">Lista desplegable con error:</label>
  <div class="controls">
	<select id="select-error">
		<option>opcion</option>
		<option>2</option>
		<option>3</option>
		<option>4</option>
		<option>5</option>
	</select>
	<div class="help-block">Corregir el error</div>
  </div>
</div>
