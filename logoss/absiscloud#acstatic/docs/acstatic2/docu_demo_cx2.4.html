Text Area

Estilo de elementos de Área de texto

Para tener una entrada de área de texto obligatoria tendremos que añadir el estilo required al de form-group a una entrada de área de texto estándard. 


<div class="form-group required">
<label for="textarea-required" class="control-label">Área de texto*:</label>
  <div class="controls col-md-4">
	<textarea rows="3" id="textarea-required" class="input-xlarge"></textarea>
  </div>
</div>


Para un campo de área de texto inhabilitado, a parte del atributo disabled en textarea, se deberá añadir el estilo disabled al de form-group. 


<div class="form-group disabled">
  <label for="textarea-disabled" class="control-label">Área de texto inhabilitada:</label>
  <div class="controls col-md-4">
	<textarea rows="3" id="textarea-disabled" disabled="disabled" class="input-xlarge"></textarea>
  </div>
</div>


Corregir el error

Para mostrar un error, se deberá añadir el estilo error al de form-group. Si se quiere añadir un mensaje aclaratorio, el texto deberá ser del estilo help-block. 


<div class="form-group error">
  <label for="textarea-error" class="control-label">Área de texto con error:</label>
  <div class="controls col-md-4">
	<textarea rows="3" id="textarea-error" class="input-xlarge"></textarea>
	<div class="help-block">Corregir el error</div>
  </div>
</div>
[texto de ayuda]

Para mostrar el texto de ayuda añadiremos un div al final del textarea con la classe block-hint. El texto deberá ir entre corchetes [][]. 


<div class="form-group ">
  <label for="textarea-error" class="control-label">Área de texto de ayuda:</label>
  <div class="controls col-md-4">
	<textarea rows="3" id="textarea-error" class="input-xlarge"></textarea>
	<div class="help-block">Corregir el error</div>
  </div>
  <div class="block-hint empty-label-margin">[texto de ayuda]</div>
</div>

Para añadir un campo de texto enriquecido simplemente añadiremos la classe css rich-text-area al campo de texto en cuestión. 


<div class="form-group ">
  <label for="textarea-error" class="control-label">Área de texto enriquecido:</label>
  <div class="controls col-md-4">
	<textarea rows="3" id="textarea-error" class="rich-text-area"></textarea>
  </div>
</div>
