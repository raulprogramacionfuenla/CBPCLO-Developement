Filtros

Togglebox Simple

El div principal contiene la clase css toggle-box.

El botón que sirve para contraer y expandir se situará como primer hijo de este div. Será un elemento a con la clase toggle-box-boton-contraer y el atributo data-toggle="collapse". El atributo data-contraer indicará el texto a mostrar cuando el botón sea de contraer, y el atributo data-expandir para cuando sea de expandir. Además, con el atributo href indicaremos el identificador del elemento a expandir/contraer.

El contenido se define en un div con la clase panel-collapse collapse. Si añadimos la clase in indicaremos que el elemento se mostrará expandido inicialmente. Dentro de ese div pondremos el contenido en otro div con la clase panel-body. El formulario deberá tener las clases css form-horizontal inline-fields. Dentro del formulario se ubican los campos siguiendo las estructuras y normativas generales del los campos definidas en esta misma página.

Para las ubicaciones de los campos y para definir la amplitud de los mismos se puede usar el sistema de "col-X" definido en el apartado Sistema Grid

El boton de envio se pone directamente al final de formulario con una clase css btn. 


<div id="toggle-box-simple" class="form-group-inner">

	<div class="toggle-box">
		<a class="toggle-box-boton-contraer" data-toggle="collapse" href="#toggle-simple" data-contraer="Ocultar Filtros" data-expandir="Ver Filtros"></a>
	        
		<div id="toggle-simple" class="panel-collapse collapse in">
			<div class="panel-body">
				<form class="form-horizontal inline-fields ">
					...		
					<button title="Título del botón" class="btn" type="submit">Filtrar</button>		
				</form>											
			</div>
		</div>
	</div>

</div>

Togglebox super expandido

Más parámetros Menos parámetros

El div principal contiene la clase css toggle-box y su estructura es la misma que en el toogle simple, con el añadido de que si se pone la clase completo se muestra todo expandido.

Si al formulario le añadimos la clase css super-expandido al expandirse se renderizará con un campo en cada fila.

El contenido del formulario de filtros se define en un div con la clase toggle-box-content. El formulario deberá tener las clases css form-horizontal inline-fields. Dentro del formulario se ubican los campos siguiendo las estructuras y normativas generales del los campos definidas en esta misma página. Para las ubicaciones de los campos y para definir la amplitud de los mismos se puede usar el sistema de "spanX" definido en el apartado Sistema Grid

El boton de envio se pone directamente al final de formulario con una clase css btn.

Los campos que se quieren mostrar/ocultar con el boton de mas/menos parametros deberemos ponerles la clase css opcional.

Los botones que sirven para mostrar/ocultar se ubicarán justo después de cerrar el formulario y se formarán con un span que tenga las clases css toggle-box-boton-completar y toggle-box-boton-resumir respectivamente. 


<div class="toggle-box completo">
	<a class="toggle-box-boton-contraer" data-toggle="collapse" href="#toggle-super" data-contraer="Ocultar Filtros" data-expandir="Ver Filtros"></a>

	<div id="toggle-super" class="panel-collapse collapse in">
		<div class="panel-body">
			<div class="toggle-box-content">
				<form class="form-horizontal inline-fields super-expandido">
					<div class="inline-fields disabled opcional form-group">
						<label for="disabledInput" class="control-label">Entrada inhabilitada:</label>
						<div class="controls">
							<input type="text" id="disabledInput" value="inhabilitado" disabled="disabled" class="input-medium form-control">
						</div>
					</div>
					<div class="required form-group">
						<label for="focusedInput" class="control-label">Entrada obligatoria*:</label>
						<div class="controls">
							<input type="text" value="" class="input-medium form-control">
						</div>
					</div>
					<div class="form-group">
						...
					</div>
					<div class="opcional form-group">
						...
					</div>
					<div class="radiobutton-group col-sm-2 col-md-2 form-group">
						...        	
					</div>
						<button title="Título del botón" class="btn" type="submit">Filtrar</button>
				</form>
				<span class="toggle-box-boton-completar">Más parámetros</span>
				<span class="toggle-box-boton-resumir">Menos parámetros</span>
			</div>
		</div>
   </div> 
</div>
