Radio butons y checkboxes

Checkboxes y Radionbuttons



Para tener un grupo de checkbox alineados se deberá primero informar que es un grupo añadiendo el estilo checkbox-group a la div con estilo form-group. Después se deberá añadir el estilo inline al label de cada checkbox. 


<div class="form-group checkbox-group">
  <label for="inlineCheckbox1" class="control-label">Checkboxes alineados</label>
  <div class="controls">
    <label class="checkbox inline">
      <input type="checkbox" value="option1" id="inlineCheckbox1"> 1
    </label>
    <label class="checkbox inline">
      <input type="checkbox" value="option2" id="inlineCheckbox2"> 2
    </label>
    <label class="checkbox inline">
      <input type="checkbox" value="option3" id="inlineCheckbox3"> 3
    </label>
  </div>
</div>


Para tener un grupo de checkbox en columna se deberá informar que es un grupo añadiendo el estilo checkbox-group a la div con estilo form-group. 


<div class="form-group checkbox-group">
  <label class="control-label">Checkboxes en columna</label>
  <div class="controls">
    <label class="checkbox">
      <input type="checkbox" value="option1" name="optionsCheckboxList1">
      Opción 1
    </label>
    <label class="checkbox">
      <input type="checkbox" value="option2" name="optionsCheckboxList2">
      Opción 2
    </label>
    <label class="checkbox">
      <input type="checkbox" value="option3" name="optionsCheckboxList3">
      Opción 3
    </label>
     <label class="checkbox disabled">
         <input disabled="disabled" value="option1"  type="checkbox" />
       Checkbox inhabilitado
     </label>
     <label class="checkbox error">
         <input value="option1"  type="checkbox" />
       Checkbox con error
        <div class="help-block">Corregir el error </div>
     </label>
  </div>
</div>


Para tener un grupo de radio-button se deberá informar que es un grupo añadiendo el estilo radiobutton-group al div con estilo form-group. Cada elemento radio será un div con la clase radio. 


<div class="radiobutton-group form-group">
  <label class="control-label">Radio buttons</label>
  <div class="controls">
    <div class="radio"><label class=""> <input type="radio" checked="checked" value="option1" id="optionsRadios1" name="optionsRadios"> 
      Opción A
    </label></div> 
    <div class="radio"><label class="disabled"> <input type="radio" value="option2" id="optionsRadios2" disabled="disabled" name="optionsRadios">
      Opción B
    </label></div>
    <div class="radio"><label class="error"> <input type="radio" value="option2" id="optionsRadios2" name="optionsRadios">
      Opción C
      <div class="help-block">Corregir el error</div>
      </label></div>
  </div>
</div>


Para un checkbox inhabilitado, a parte del atributo disabled en el input, se deberá añadir el estilo disabled al label. 


<div class="checkbox-group form-group">
  <div class="controls">
    <div class="checkbox">
      <label class="disabled">
        <input disabled="disabled" value="option1" type="checkbox">
        Checkbox inhabilitado
      </label>
    </div>
  </div>
</div>


Para un checkbox con error se deberá añadir el estilo error al label. 


<div class="checkbox-group form-group">
  <div class="controls">
    <div class="checkbox">
      <label class="error">
        <input value="option1" type="checkbox">
        Checkbox con error
      </label>
    </div>
  </div>
</div>


Para un checkbox de salida se deberá añadir el estilo consultas-controls al elemento div padre del elemento ul.

Para cada elemento input de tipo checkbox deberá añadir el estilo consulta 


<div class="form-group radiobutton-group">
  <label class="control-label">Checkboxes de salida</label>
  <div class="consultas-controls">
    <ul>
      <li>
        <label class="checkbox">  
          <input class="consulta" type="checkbox"
          value="option1" name="optionsCheckboxList1">
          Opción 1
        </label> 
      </li>
      <li>
        <label class="checkbox">
          <input class="consulta" type="checkbox"
          value="option2" checked="checked" name="optionsCheckboxList2">
          Opción 2
        </label>
      </li>
      <li> 
        <label class="checkbox">
          <input class="consulta" type="checkbox"
          value="option3" checked="checked" name="optionsCheckboxList3">
          Opción 3
        </label>
      </li>
      ...
    </ul>
  </div>
</div>
