Estructura del widget

Estructura del widget patrón horizontal normal (4/4 = 388px) y patrón vertical 1/4 (91px)

Cabecera que generará el contenedor

El lay-out anterior es el mismo para todos estos widgets, sin importar sus dimensiones. Lo que cambia son las clases adicionales que establecen dichas dimensiones. Para este caso:

Para establecer patrón horizontal normal (4/4 = 388px) se usa de la clase widget-inner-hpattern-4-4.

Para establecer patrón vertical 1/4 (91px) se usan las clases widget-inner-vpattern-1-4 y widget-body-vpattern-1-4. 


<div class="widget-inner widget-inner-hpattern-4-4 widget-inner-vpattern-1-4">
	<div class="widget-header">
	</div>
	<div class="widget-body widget-body-vpattern-1-4" >
		Lorem ipsum dolor sit amet, consectetur adipiscing elit...
	</div>
	<div class="widget-footer">
	</div>
</div>
