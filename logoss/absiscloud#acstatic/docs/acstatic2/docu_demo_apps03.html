Enlace con desplegable de selección única

Cabecera que generará el contenedor

Para hacer aparecer un enlace con un desplegable de selección única utilizaremos un tag <span> con clase flexdropdownmenu-button. Informaremos en dicho tag los atributos siguientes:

data-unica con valor y.

data-offsets con valor 0,-1.

data-flexmenu con un valor que se corresponderá con al identificador que asignaremos a la lista de valores del desplegable. 


<div class="widget-control-group-content">
	<div class=dropdown-link">
		<label>Categoría: </label>
		<div class="flexdropdownmenu-button"
				data-flexmenu="flexmenu"
				data-unica="y" data-offsets="0,-1">
			<span class="selection">Notarios (3)</span>
			<i class="icono-desplegable-simple-gran"></i>
		</div>	
		<ul id="flexmenu" class="flexdropdownmenu">
			<li><i class="icono-ico-unassigned_c"></i><span>Sin categoria (1)</span></li>
			<li><i class="icono-ico-yellow"></i><span>Personal (5)</span></li>
			<li><i class="icono-ico-blue"></i><span>Clientes (1)</span></li>
			<li><i class="icono-ico-green"></i><span>Notarios (3)</span></li>
			<li class="bold-type"><i class="icono-ico-red"></i><span>Urgente (2)</span></li>
			<li class="bold-type"><i class="icono-ico-envelope"></i><span>Recibidas (1)</span></li>
		</ul>
	</div>
</div>

La lista de valores del desplegable la informaremos en un tag <ul> al que asignaremos clase flexdropdownmenu. Los elementos de la lista se definen mediante tags <li> estándar. Cada uno de los elementos puede llevar un icono a su izquierda. También es posible indicar que un elemento debe mostrarse en negrita asignándole clase bold-type.
