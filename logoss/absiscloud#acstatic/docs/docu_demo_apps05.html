Listado de elementos

Cabecera que generará el contenedor

Para hacer aparecer un listado de elementos crearemos un DIV con clases widget-container y items-list. Las elementos de las filas del listado se definen dentro de este DIV utilizando los tags estándar <ul> y <li>.

Cada fila del listado contiene texto dentro de un <span>, bien con peso regular o en negrita. Para hacer aparecer la negrita asignaremos al <span> clase bold-type.

Opcionalmente puede mostrarse un icono a la izquierda en cada fila.

En algunos casos puede ser necesario habilitar a la derecha de cada fila un desplegable. Procederemos de manera similar a la explicada en la página Enlace con desplegable de selección única, pero en lugar de aplicarlos sobre un DIV, lo aplicaremos sobre un tag i. A este tag le asignaremos dos clases, icono-desplegable-simple-gran y items-list-dropdown. Además, en su atributo data-dir indicaremos valor h. 


<div class="widget-control-group-content">
	<div class="widget-container items-list">
		<ul>
			<li>
				<i class="icono-ico-envelope items-list-icon"></i>
				<span class="bold-type">Las notas recibidas se resaltan en negrita, y se les asigna el icono de sobre.</span>
				<i class="icono-desplegable-simple-gran items-list-dropdown" data-flexmenu="flexmenu2" data-dir="h"></i>
			</li>
			<li>
				<i class="icono-ico-red items-list-icon"></i>
				<span>El estilo tipográfico de las notas una vez leídas es regular</span>

			</li>
