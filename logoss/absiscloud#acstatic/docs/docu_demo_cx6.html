Estructura de página

Estructura general

Dentro del body de una página estandard encontramos una estructura similar a la siguiente. Destacar que el contenido se divide en dos div, Una con span9 para el contenido y sus secciones y otra con span3 para el menú. 


<div class="container-fluid">
  <div class="content">
		
    <!-- Page Title -->
    <div class="page-header"> 
      ...
    </div> 
	     
    <!-- Content  -->
    <div class="row-fluid">
      <div class="span12">
        <div class="row-fluid">
          <div class="span9">
            <div class="row-fluid">
              <div class="span12">
            		
                <div class="control-group">
                  <h2>Estructura general</h2>
                  <div class="control-group">
                    Contenido sección.
                  </div>
                </div>
                 ...	            
              </div>       
            </div>
          </div>
	  
          <!-- Menu -->
          <div id="menu-tf7-right" class="span3">
            ...
          </div>
        </div>
      </div>  
    </div><!--/row-->
  </div><!--/ .content-->
</div><!--/.fluid-container-->

<footer>
  ...
</footer>

<!-- Javascripts -->
...

Cabecera de página

La cabecera de la página está contenida en un div con estilo control-group. El título deberá ser un h2. 


<div class="page-header"> 
  <h1>Elementos de Página</h1>
</div>

Menú lateral

El menú estará contenido en un div de estilo menu-tf7-right con otro div anidado con estilo sidebar-nav. Para listar las diferentes páginas usaremos un elemento de lista ul con estilos nav nav-list. 


<div id="menu-tf7-right" class="span3">
  <div class="sidebar-nav">
    <ul class="nav nav-list">
      <li><a href="docu_demo_cx1.html" class="current" data-original-title=""> Elementos de Navegación</a></li>
      <li><a href="docu_demo_cx2.html" data-original-title=""> Formularios</a></li>
      <li><a href="docu_demo_cx3.html" data-original-title=""> Tablas y Listas</a></li>
      <li><a href="docu_demo_cx4.html" data-original-title=""> Mensajes</a></li>
      <li><a href="docu_demo_cx5.html" data-original-title=""> Sistema Grid</a></li>
    </ul>
  </div>
</div>

Sección

Para las distitas secciones usaremos un div con el estilo control-group. Para poner el título con la barra de separación usaremos una etiqueta h2. Para el contenido justificado un div con estilo control-group-inner 


<div id="id sección" class="control-group">
  <h2>Título sección</h2>
  <div class="control-group-inner">	 
  ...
  </div>
</div>

Pie de página

El pie de página debe definirse en un elemento footer. Deberemos incluir el separador hr y el contenido informativo. 


<div class="footer">
